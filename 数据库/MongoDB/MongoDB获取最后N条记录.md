# MongoDB获取最后N条记录

## 采用`_id`字段进行排序

自动创建的`_id`字段，它有一个日期嵌入其中，可以使用它来排序进行排序。

### 升序排列（最旧到最新）

这将返回从最旧到最新排序的所有文档。

```
db.foo.find().sort({"_id":1})
```

### 降序排列（最新到最旧）

这将返回从最新到最旧排序的所有文档。

```
db.foo.find().sort({"_id":-1})
```


## 采用`.limit()`方法进行限制

当做广泛的查询，可以添加一个限制。

```
db.foo.find().sort({"_id":1}).limit(50)
```
